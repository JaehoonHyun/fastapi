
### pre requirement
python version > 3.6
pip3
```
apt-get install python3-pip
```

### setting venv
```
apt-get install python3-venv
python3 -m venv env
```


### how to install python library
```
python3 -m pip install fastapi
pip3 install uvicorn
```

### how to execute
```
uvicorn main:app --reload
```
